# gamemode_dm

Nothing special Just a dm Gamemode for FiveM 

#### Requirement :

[Myhtic Notify]: https://forum.cfx.re/t/dev-resource-mythic-notifications/587071

#### Features : 

Select a team with `/team [t - ct] `

- T : Hamajistan
- Ct : SuperCops

Spawn a car with `/car`

Other usefull cmds : 
`/help - /respawn`
